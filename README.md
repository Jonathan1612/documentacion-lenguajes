# 🎓 DevDocs Learning Platform

Plataforma interactiva de aprendizaje de programación con documentación completa, ejemplos prácticos y ejercicios validados automáticamente.

## ✨ Características

- 📚 **Documentación progresiva**: De básico a experto
- 🔍 **Búsqueda inteligente**: Encuentra cualquier concepto rápidamente
- 💻 **Editor integrado**: Practica código en el navegador
- ✅ **Ejercicios validados**: Sistema automático de verificación
- 🎯 **Progreso tracking**: Sigue tu avance
- 🌙 **Modo oscuro**: Cuida tus ojos

## 🚀 Lenguajes Disponibles

- ☕ **Java** - Variables, Collections, Streams, Spring Boot, Maven
- 🟨 **JavaScript** - ES6+, Async/Await, React, Node.js
- 🐍 **Python** - Básico, POO, Django, FastAPI
- 🔵 **Go** - Concurrencia, APIs, Microservicios
- 🎨 **CSS** - Flexbox, Grid, Animations
- 📄 **HTML** - Semántica, Accesibilidad, SEO

## 🛠️ Stack Tecnológico

- **Frontend**: Next.js 14 + TypeScript
- **Styling**: Tailwind CSS
- **Editor**: Monaco Editor (VS Code)
- **Syntax Highlighting**: Prism.js
- **Search**: Fuse.js

## 📦 Instalación

```bash
# Instalar dependencias
npm install

# Modo desarrollo
npm run dev

# Build para producción
npm run build
npm start
```

## 📁 Estructura del Proyecto
