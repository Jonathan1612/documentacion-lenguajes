# 🗺️ Ruta de Aprendizaje Java

Esta guía te muestra el **orden recomendado** para estudiar Java desde cero hasta nivel experto.

## 🎯 Cómo Usar Esta Guía

- ✅ **Sigue el orden numerado** - cada tema se construye sobre el anterior
- ⏱️ **Duración estimada** por tema para planificar tu estudio
- 🔄 **Practica ejercicios** antes de avanzar al siguiente tema
- 📝 **Revisa temas relacionados** cuando se mencionen

---

## 📚 NIVEL BÁSICO (1-2 meses)

### Fase 1: Fundamentos Iniciales

#### 1️⃣ [Introducción a Java](./basico/introduccion) ⏱️ 20min
**¿Qué aprenderás?**
- Tu primer programa "Hola Mundo"
- Estructura básica de un programa Java
- Compilar y ejecutar código

**✅ Requisitos:** Ninguno  
**🎯 Objetivo:** Escribir y ejecutar tu primer programa

---

#### 2️⃣ [Variables y Tipos de Datos](./basico/variables) ⏱️ 30min
**¿Qué aprenderás?**
- 8 tipos primitivos (int, double, boolean, etc.)
- Declarar y asignar variables
- String y sus métodos principales

**✅ Requisitos:** Introducción  
**🎯 Objetivo:** Manejar datos de diferentes tipos

---

#### 3️⃣ [Operadores](./basico/operadores) ⏱️ 25min
**¿Qué aprenderás?**
- Aritméticos: +, -, *, /, %
- Comparación: ==, !=, <, >, <=, >=
- Lógicos: &&, ||, !
- Ternario: ? :
- Precedencia de operadores

**✅ Requisitos:** Variables  
**🎯 Objetivo:** Realizar operaciones y comparaciones

---

#### 4️⃣ [Control de Flujo](./basico/control-flujo) ⏱️ 25min
**¿Qué aprenderás?**
- if/else para decisiones
- switch para múltiples opciones
- for, while, do-while para repetir código
- break y continue

**✅ Requisitos:** Operadores  
**🎯 Objetivo:** Controlar el flujo del programa

---

### Fase 2: Estructuras y Organización

**¿Qué aprenderás?**
- Crear y manipular arrays

**✅ Requisitos:** Control de flujo  
**🎯 Objetivo:** Manejar colecciones de datos básicas

---

#### 6️⃣ [Funciones y Métodos](./basico/funciones) ⏱️ 30min
**¿Qué aprenderás?**
- Crear y llamar métodos
- Parámetros y valores de retorno
- Sobrecarga de métodos
- Recursividad básica

**✅ Requisitos:** Arrays  
**🎯 Objetivo:** Organizar código en funciones reutilizables

---

#### 7️⃣ [Strings Avanzado](./basico/strings) ⏱️ 25min
**¿Qué aprenderás?**
- Métodos de String
- StringBuilder y StringBuffer
- Formateo de strings
- Expresiones regulares básicas

**✅ Requisitos:** Funciones

---

#### 8️⃣ [POO - Clases y Objetos](./basico/poo-clases) ⏱️ 35min
**¿Qué aprenderás?**
- Crear clases y objetos

#### 9️⃣ [Entrada y Salida Básica](./basico/entrada-salida) ⏱️ 20min
**¿Qué aprenderás?**

- Formateo de salida (printf)

## 🎯 Consejos para el Aprendizaje

### ✅ Hacer

1. **Practica cada tema** - Escribe código, no solo leas
2. **Resuelve los ejercicios** - Todos los temas tienen ejercicios
3. **Crea proyectos pequeños** - Aplica lo aprendido
4. **Lee código de otros** - GitHub, StackOverflow
5. **Revisa temas anteriores** - Reforzar es importante

### ❌ Evitar

1. **No saltar temas** - La secuencia es importante
2. **No memorizar** - Entiende los conceptos
3. **No ver solo teoría** - Practica es clave
4. **No avanzar sin entender** - Mejor ir lento pero seguro
5. **No estudiar todo de golpe** - Toma descansos
