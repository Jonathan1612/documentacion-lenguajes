## 🚀 Cómo Empezar

1. **Completa Python básico/intermedio/avanzado** primero
2. Lee el [AI Engineering Roadmap](../AI_ENGINEERING_ROADMAP.md)
3. Sigue los documentos en orden (SQL → APIs → FastAPI → etc.)
4. Construye cada proyecto del portfolio
5. Únete a comunidades (LangChain Discord, AI Engineer Summit)

## 💡 Tips

- **Para desarrollo**: ChromaDB (local), sqlite, OpenAI
- **Para producción**: Pinecone/Weaviate, PostgreSQL, Railway/Render
- **Costos**: Empieza con gpt-3.5-turbo, migra a gpt-4 solo cuando lo necesites
- **Testing**: Siempre testea prompts con múltiples inputs antes de producción
